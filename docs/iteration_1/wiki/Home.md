# Карьерный навигатор по компетенциям

### О проекте
Веб-приложение для построения индивидуальных траекторий обучения IT-профессиям. Помогает студентам и специалистам ответить на вопрос: «Что и в каком порядке учить, чтобы стать ___?».

### Проблема
Студентам и начинающим специалистам сложно самостоятельно определить оптимальный порядок изучения навыков для выбранной профессии. Информации много и она разрознена.

### Решение
Инструмент, который на основе данных из различных источников (образовательные платформы, сайты вакансий, открытые базы знаний) формирует граф зависимостей навыков и рекомендует учебные материалы в логической последовательности.
