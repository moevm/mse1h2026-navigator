# mse1h2026-navigator
## О проекте
### Постановка задачи

**Задача**: Помочь студентам и специалистам строить индивидуальные траектории обучения для целевых профессий. Например, студент 3 курса по специальности «Прикладная математика и информатика» хочет стать ML Engineer, ему нужно увидеть весь граф необходимых навыков (от математики до конкретных фреймворков), найти курсы, проекты и темы для изучения в оптимальном порядке.

**Функциональность**:
- поиск подграфа по различным критериям: профессия, навык, курс, технология, компания;
- автоматическое построение графа зависимостей "для освоения навыка X нужны навыки Y и Z";
- рекомендации ресурсов: MOOCs (Coursera, edX, OpenEdu, Stepic  и др.), статьи, книги из Wikidata/DBpedia;
- построение отчета: "Целевая профессия - Ключевые навыки - Предварительные навыки/знания - Ресурсы для изучения";
- оценка времени изучения;
- задание приоритетов;
- экспорт результатов.

**Технологии**:
- Wikidata (профессии, навыки) + внешние API образовательных платформ;
- GraphQL поверх графа для гибких запросов;
- React Flow для интерактивного построения дорожных карт;
- Python (алгоритмы топологической сортировки для оптимального порядка изучения).

**Результат**: веб-приложение, где пользователь выбирает целевую роль/проект, получает интерактивный граф компетенций с ресурсами, может корректировать путь и экспортировать персональный учебный план в удобный формат.

### Запланированные и выполненные задачи
- Создание wiki страницы с описанием требований, макета UI и сценариями использования
